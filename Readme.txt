Mini Projet de Lucas MARROT et Kyliann CLARET-LAVAL

Compilation :
gcc -o server server.c
gcc -o client client.c

Exécution :
./server
./client

Commandes disponibles :
/help : Afficher l'aide

/quit : Quitter le chat
/switch [channel] : Changer de channel
/send [file] : Envoyer un fichier au channel
/download [file] : Télécharger un fichier du channel
/list : Lister les fichiers du channel
